О проекте

Данный проект является дипломной работой курса по направлению «Создание web-приложений с использованием Python».

При разработке данного проекта были использованы Python, Django Framework, СУБД (PostgreSQL), HTML/CSS, Bootstrap, API Яндекс.Карт, Telegram API.

Главная цель проекта?

Перед вами рекомендации и краткие сведения о спортивных площадках различных видов спорта.

Как добавить площадку?

Для добавления новой площадки просто перейдите по ссылке "Добавить площадку" и следуйте инструкциям.
